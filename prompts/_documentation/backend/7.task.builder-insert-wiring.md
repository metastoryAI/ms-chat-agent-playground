# BUILDER Insert Wiring — Frontend Task

## Sorun

Agent-chat BUILDER response'u (modules + features + subfeatures, summary + assumptions dolu) frontend store'da tutuluyor ama **DB'ye yazılmıyor**. `builder_insert` case'i sadece toaster ve UI state update yapıyor — hiçbir API call yok.

**Referans:** [agent-chat.store.ts:819](../../ms-ui/src/modules/boards/stores/agent-chat.store.ts)

## Bağımlılık

Backend yeni endpoint açacak:
```
POST /agent/commit-builder-tree
Body: currentStructure (agent-chat LLM shape — name, summary[], assumptions, ...)
```

Detay: [../backend/5.task.agent-commit-endpoint.md](../backend/5.task.agent-commit-endpoint.md)

Backend endpoint hazır olana kadar bu task merge edilmemeli.

## Yapılacaklar

### 1. API tanımı

`ms-ui/src/api/agent.apis.ts` (veya uygun yer):

```ts
async commitBuilderTree(
  data: { workspaceId: string; projectId: string; structure: AgentNs.Structure }
): Promise<FeatureNs.Feature[]> {
  const url = `${this.apiUrl}/agent/commit-builder-tree`;
  return httpPost({ url, data, headers: getHeaders() });
}
```

### 2. `builder_insert` case wiring

[agent-chat.store.ts:819](../../ms-ui/src/modules/boards/stores/agent-chat.store.ts):

**Şu an:**
```ts
case 'builder_insert': {
  // toaster + UI state only
  this.addMessage('assistant', text, tag);
}
```

**Olması gereken:**
```ts
case 'builder_insert': {
  if (!this.currentStructure?.sections?.length) return;

  await agentApi.commitBuilderTree({
    workspaceId: this.projectState.workspace_id,
    projectId:   this.projectState.project_id,
    structure:   this.currentStructure,
  });

  // mevcut toaster + UI state update
  this.addMessage('assistant', text, tag);
  // feature store refresh (tree'yi yeniden çek)
  await featureStore.getAllFeatures({ workspaceId, projectId });
}
```

### 3. Diff commit (updated_modules) — ileride

`currentStructure.diff` true ise farklı endpoint'e gitmek gerekebilir
(`POST /agent/commit-builder-diff`). Backend kararına bağlı.
Bu task kapsamı **dışında**.

## Testler

**T1 — happy path:**
- Given: `currentStructure.sections` dolu
- When: `builder_insert` action
- Then: `commitBuilderTree` çağrılır, structure payload as-is gider

**T2 — boş structure:**
- Given: `sections: []`
- Then: API call yok, erken return

**T3 — API hatası:**
- Given: endpoint 500 döner
- Then: toaster error, UI state değişmez, user retry yapabilir
