# Agent Commit Endpoint — Backend Task

## Sorun

BUILDER LLM response'u (modules + features + subfeatures, summary + assumptions dolu) frontend store'da tutuluyor ama DB'ye yazılmıyor. Mevcut `/feature/add-groups-with-features` endpoint'i DB-shape payload bekliyor — frontend'in LLM→DB mapping yapması gerekiyor, bu da abstraction leak.

## Çözüm

Backend yeni endpoint açar, **LLM-shape'i as-is kabul eder**, iç mapping uygular.

```
POST /agent/commit-builder-tree
```

Request body — frontend'in elindeki `currentStructure`:

```json
{
  "workspaceId": "...",
  "projectId": "...",
  "structure": {
    "sections": [
      {
        "id": "...",
        "name": "Core",
        "modules": [
          {
            "id": "mod_xyz",
            "name": "Authentication",
            "summary": [{ "text": "User login and signup flows.", "source": "doc_1" }],
            "assumptions": ["OAuth providers not specified."],
            "color": "#4A90E2",
            "features": [
              {
                "id": "feat_abc",
                "name": "Login",
                "summary": [{ "text": "Email + password authentication.", "source": "doc_1" }],
                "assumptions": ["MFA not confirmed."],
                "subfeatures": [
                  {
                    "id": "subfeat_123",
                    "name": "Email Login",
                    "summary": "Authenticate via email and password."
                  }
                ]
              }
            ]
          }
        ]
      }
    ]
  }
}
```

Response: `List<Feature>` (mevcut `addGroupsWithFeatures` ile aynı dönüş şekli).

## DB Schema Gap (Prerequisite)

`feature` tablosuna `assumptions` JSONB kolonu ekleme:

### Flyway migration
```sql
ALTER TABLE feature
  ADD COLUMN assumptions jsonb;
```

### Entity update
[Feature.java](../../../fp-api/project/src/main/java/io/featureplanner/model/jpa/Feature.java):
```java
@Column(name = "assumptions", columnDefinition = "jsonb")
@JdbcTypeCode(SqlTypes.JSON)
private List<String> assumptions;
```

Subfeature'da assumptions yok (prompt üretmiyor) — kolon her satır için nullable.

## LLM → DB Mapping (Backend İçinde)

Contract: [0.spec.agent-to-db-mapping.md](0.spec.agent-to-db-mapping.md)

Özet:

| LLM alanı | DB kolon | Dönüşüm |
|---|---|---|
| `name` | `summary` | substring(0, 255) |
| `summary[0].text` | `description` | flatten (plain string subfeature için as-is) |
| `assumptions` | `assumptions` | as-is (jsonb) |
| `color` | `properties.color` | properties map'e yerleştir |
| Module | `feature_type = GROUP` | — |
| Feature | `feature_type = FEATURE` | — |
| Subfeature | `feature_type = SUB_FEATURE` | — |

## Implementation

### Yeni DTO'lar

`AgentCommitRequest.java`:
```java
public record AgentCommitRequest(
    Long workspaceId,
    Long projectId,
    StructureDto structure
) {}

public record StructureDto(List<SectionDto> sections) {}
public record SectionDto(String id, String name, List<ModuleDto> modules) {}

public record ModuleDto(
    String id,
    String name,
    List<SummaryEntryDto> summary,
    List<String> assumptions,
    String color,
    List<FeatureDto2> features
) {}

public record FeatureDto2(
    String id,
    String name,
    List<SummaryEntryDto> summary,
    List<String> assumptions,
    List<SubfeatureDto> subfeatures
) {}

public record SubfeatureDto(
    String id,
    String name,
    Object summary   // String | List<SummaryEntryDto> — polymorphic
) {}

public record SummaryEntryDto(String text, String source, String date) {}
```

### Controller

`AgentCommitController.java`:
```java
@PostMapping("/agent/commit-builder-tree")
public List<Feature> commit(@RequestBody AgentCommitRequest req) {
    return agentCommitService.commit(req);
}
```

### Service — mapping + persist

`AgentCommitService.commit()`:
```java
// 1. Walk tree → Feature entity'lere çevir
// 2. GROUP → FEATURE → SUB_FEATURE sıralı repository.save()
// 3. parent_id bağlantılarını gerçek UUID'lerle kur
// 4. Event publish (mevcut FeatureService pattern'i)
// 5. Kaydedilen listeyi dön
```

Mapping helper'ları:
```java
private String toDbTitle(String name) {
    if (name == null) return "";
    return name.length() > 255 ? name.substring(0, 255) : name;
}

private String flattenSummary(Object summary) {
    if (summary == null) return null;
    if (summary instanceof String s) return s;
    if (summary instanceof List<?> list && !list.isEmpty()) {
        var first = (Map<?, ?>) list.get(0);
        return (String) first.get("text");
    }
    return null;
}
```

### Mevcut `FeatureService` kullanımı

Reinvent etme — `FeatureService.saveRecursively()` veya benzeri mevcut factory'leri kullan. Yeni service sadece **adaptation layer**: LLM tree → FeatureDto list → mevcut save flow.

## Testler (Given-When-Then)

**T1 — Happy path commit:**
- Given: 1 section, 2 module, her modülde 2 feature, 1 feature'da 1 subfeature
- When: `POST /agent/commit-builder-tree`
- Then: 7 feature satırı DB'de (2 GROUP + 4 FEATURE + 1 SUB_FEATURE)
- Then: her biri doğru `feature_type`, `summary` (title), `description`, `assumptions` ile

**T2 — Assumptions persist:**
- Given: module assumptions = `["A1", "A2"]`
- When: commit
- Then: DB row `assumptions` jsonb = `["A1", "A2"]`

**T3 — Subfeature assumptions atılır:**
- Given: subfeature payload'da yanlışlıkla `assumptions` alanı var
- Then: DB'ye yazılmaz (subfeature'da alan yok — silent drop)

**T4 — Long name truncate:**
- Given: `name` 300 karakterlik
- Then: DB `summary` = ilk 255 karakter

**T5 — Empty summary array:**
- Given: module.summary = `[]`
- Then: DB `description` = null

**T6 — Polymorphic subfeature summary:**
- Given: subfeature.summary = `"plain string"` (subfeature prompt şekli)
- Then: DB `description` = `"plain string"` (flatten gereksiz)

**T7 — Parent ID resolution:**
- Given: synthetic ID'ler (`mod_abc`, `feat_xyz`) parent reference'larında
- Then: Save sırasında synthetic → real UUID map'lenir, `parent_id` gerçek UUID'ye set edilir

**T8 — Unauthorized workspace:**
- Given: user başka workspace'in projesine commit atmaya çalışır
- Then: 403 Forbidden (mevcut security pattern)
